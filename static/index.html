Acerca del Proyecto

Science Agent es una herramienta de investigación académica desarrollada para el DataHack 2026 · Reto 3. Combina un agente LangGraph con modelos de lenguaje Ollama para automatizar la búsqueda y síntesis de literatura científica.

El sistema busca simultáneamente en ArXiv y Google Scholar, construye ecuaciones booleanas elaboradas con sinónimos y operadores de campo, y genera matrices bibliográficas con análisis de tendencias, brechas y recomendaciones de lectura.

Todo funciona completamente en local — sin API keys externas — usando modelos como llama3.1:8b a través de Ollama.

👥 Acerca de Science Agent

Transformando datos bibliográficos en conocimiento accionable

¿Qué es Science Agent?

Science Agent es una plataforma de investigación académica impulsada por inteligencia artificial, desarrollada como parte del DataHack 2026 · Reto 3. Automatiza el proceso completo de revisión de literatura científica: desde la construcción de ecuaciones de búsqueda hasta la generación de matrices bibliográficas con análisis de tendencias.

El sistema utiliza un agente LangGraph con 7 nodos orquestados, modelos locales Ollama (llama3.1:8b), y búsqueda simultánea en ArXiv y Google Scholar, todo corriendo localmente sin necesidad de API keys externas.

Objetivos

🔍
Búsqueda Automatizada
Ecuaciones booleanas con sinónimos y operadores de campo generadas por IA
📊
Síntesis Bibliométrica
Matrices con métricas cuantitativas, tendencias y recomendaciones de lectura
🕸
Visualización de Redes
Grafos de conocimiento y mapas de calor de similitud entre artículos
💻
100% Local
Sin dependencias de API externas, privacidad total de los datos de investigación

¿Cómo funciona?

1

Describe tu tema — El agente recibe tu área de investigación o pregunta científica con el mayor detalle posible.

2

Ecuación booleana — El LLM construye una ecuación elaborada con sinónimos, variantes morfológicas y operadores de exclusión.

3

Búsqueda paralela — Se consultan simultáneamente ArXiv (hasta 25 papers) y Google Scholar (hasta 25 papers).

4

Selección de plantilla — Elige entre 6 templates especializados: Estado del Arte, Revisión Sistemática, Benchmarking, Marco Teórico, Tendencias o Meta-análisis.

5

Matriz bibliográfica — El LLM genera una tabla Markdown detallada con análisis de tendencias, brechas y recomendaciones.

6

Q&A sobre papers — Haz preguntas específicas sobre la literatura encontrada; el agente responde con citas concretas.

7

Exportar a XLSX — Descarga la matriz en Excel con colores claros: artículos, matriz bibliográfica y metadatos de búsqueda.
